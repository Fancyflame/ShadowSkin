<h3>我的皮肤能保存在云端吗？</h3>
<blockquote>
当然可以！不过它并不是保存在云端，而是在您的浏览器里。
不过前提是你要保证你的浏览器有<b>localStorage</b>功能
简言之，你的浏览器版本需要足够高！
不过就算你不知道你的浏览器能不能保存也没关系，它会提醒你的
</blockquote>
